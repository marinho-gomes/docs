# Quickstart Guide

Get started with AI Image Generator and create your first AI-generated image.

## Prerequisites

<Check>Modern web browser (Chrome, Firefox, Safari, Edge)</Check>
<Check>Internet connection</Check>

## Step 1: Access the Dashboard

Navigate to the dashboard in your browser. You'll see the 3-column interface:

<Frame>
  <img src="/images/dashboard-overview.png" alt="Dashboard Overview" />
</Frame>

The layout consists of:
- **Left**: Navigation sidebar
- **Center**: Control panel
- **Right**: Canvas area

## Step 2: Select Your Model

Click the model dropdown to choose your AI model:

| Model | Best For |
|-------|----------|
| **Flux 1.0 Ultra** | Highest quality, production use |
| **Midjourney V6** | Artistic, stylized images |
| **DALL-E 3 Max** | Versatile, detailed scenes |

<Tip>
  Start with **Flux 1.0 Ultra** for the best balance of quality and speed.
</Tip>

## Step 3: Write Your Prompt

Enter a description of the image you want to create in the prompt textarea:

```text
A serene mountain landscape at sunset, with a crystal-clear lake
reflecting orange and purple clouds, pine trees in the foreground
```

<Info>
  Be specific! Include details about lighting, composition, colors, and style for better results.
</Info>

### Prompt Tips

<AccordionGroup>
  <Accordion title="Be Descriptive">
    Instead of "a cat", try "a fluffy orange tabby cat sleeping on a velvet cushion, soft window light"
  </Accordion>
  <Accordion title="Specify Style">
    Add artistic direction: "in the style of studio ghibli", "photorealistic", "oil painting texture"
  </Accordion>
  <Accordion title="Include Lighting">
    "golden hour lighting", "dramatic shadows", "soft diffused light", "neon glow"
  </Accordion>
  <Accordion title="Set the Mood">
    "peaceful and serene", "energetic and vibrant", "mysterious and dark"
  </Accordion>
</AccordionGroup>

## Step 4: Choose Style & Aspect Ratio

### Style Presets

Select a style preset to influence the visual aesthetic:

<CardGroup cols={4}>
  <Card title="Realistic">
    Photorealistic rendering
  </Card>
  <Card title="Anime">
    Japanese animation style
  </Card>
  <Card title="3D Render">
    CGI/Octane quality
  </Card>
  <Card title="Oil">
    Traditional painting look
  </Card>
</CardGroup>

### Aspect Ratios

Choose the output dimensions:

- **1:1** - Square (Instagram, profile pictures)
- **16:9** - Widescreen (desktop wallpapers, presentations)
- **9:16** - Vertical (mobile wallpapers, stories)
- **4:3** - Classic (traditional photos)
- **3:2** - Standard photo (prints)

## Step 5: Set Variant Count

Use the segmented control to choose how many image variants to generate (1-4).

<Note>
  Generating multiple variants gives you options to choose from but takes longer.
</Note>

## Step 6: Generate!

Click the purple **Generate** button. You'll see:

1. Loading overlay appears with "Dreaming..." text
2. Progress indicator shows generation status
3. Generated images appear in the canvas area

<Frame>
  <img src="/images/generation-loading.png" alt="Generation in Progress" />
</Frame>

## Step 7: View & Save Results

Once generation completes:

- **View**: Click any image to see it full-size in the canvas
- **Download**: Use the download button in the toolbar
- **Share**: Click share to get a link
- **History**: All generations are saved to your session history
