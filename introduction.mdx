# Welcome to AI Image Generator

AI Image Generator is a powerful application that transforms text prompts into stunning AI-generated images. Built with a modern tech stack and featuring an intuitive interface, it makes AI image creation accessible to everyone.

## Key Features

- **Multiple AI Models** - Choose from Flux 1.0 Ultra, Midjourney V6, DALL-E 3 Max, and more
- **Style Presets** - Apply artistic styles like Realistic, Anime, 3D Render, or Oil Painting
- **Flexible Aspect Ratios** - Generate images in 1:1, 16:9, 9:16, 4:3, and more
- **Variant Generation** - Create up to 4 image variants per prompt
- **Generation History** - Browse and manage all your past creations
- **Real-time Updates** - Watch your images generate with live status updates
