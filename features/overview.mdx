## Generation Options

### Variant Count

Generate 1-4 variations of your prompt simultaneously:

- **1 variant**: Fastest, single focused result
- **2 variants**: Quick comparison options
- **3 variants**: Good variety for selection
- **4 variants**: Maximum options, takes longest

### Reference Images

Attach reference images to guide the generation:

1. Click the attachment icon in the prompt area
2. Select an image from your device
3. The AI will use it as visual guidance

<Note>
  Reference images work best with descriptive prompts that explain how to use the reference.
</Note>

## Real-time Features

### Live Status Updates

Watch your generations progress with real-time status:

- **Pending**: Request queued
- **Processing**: AI is generating
- **Complete**: Ready to view
- **Failed**: Error occurred (with retry option)

### Session History

All generations are automatically saved to your session:

- Thumbnail strip shows recent creations
- Click any thumbnail to view full-size
- History persists across browser sessions
