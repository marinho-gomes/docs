# Components

Reference guide for all UI components used in AI Image Generator.

## Buttons

### Primary Button

The main call-to-action button.

```css
.btn-primary {
  background: linear-gradient(92deg, #8b5cf6 0%, #6366f1 100%);
  color: #FFFFFF;
  padding: 14px 24px;
  border-radius: 12px;
  font-weight: 600;
  box-shadow: 0px 0px 20px rgba(139, 92, 246, 0.5);
}
```

| State | Style |
|-------|-------|
| Default | Purple gradient with glow |
| Hover | Increased brightness |
| Active | Slight scale down |
| Disabled | 50% opacity, no glow |
| Loading | Spinner + "Generating..." |

### Secondary Button

For secondary actions.

```css
.btn-secondary {
  background: transparent;
  color: #A1A1AA;
  padding: 12px 20px;
  border: 1px solid rgba(255, 255, 255, 0.08);
  border-radius: 10px;
  font-weight: 500;
```


## Layout

### Spacing Scale

| Token | Value |
|-------|-------|
| `--space-xs` | 4px |
| `--space-sm` | 8px |
| `--space-md` | 16px |
| `--space-lg` | 24px |
| `--space-xl` | 32px |
| `--space-2xl` | 48px |

### Border Radius Scale

| Token | Value | Usage |
|-------|-------|-------|
| `--radius-sm` | 6px | Badges, small elements |
| `--radius-md` | 10px | Buttons, tabs |
| `--radius-lg` | 12px | Cards, inputs |
| `--radius-xl` | 16px | Large cards, panels |
| `--radius-2xl` | 20px | Hero sections |
| `--radius-full` | 9999px | Pills, avatars |

## Shadows

```css
:root {
  /* Elevation levels */
  --shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.2);
  --shadow-md: 0 4px 8px rgba(0, 0, 0, 0.3);
  --shadow-lg: 0 8px 32px rgba(0, 0, 0, 0.4);
```

## Animation

### Transitions

```css
:root {
  --transition-fast: 0.15s ease;
  --transition-normal: 0.2s ease;
  --transition-slow: 0.3s ease;
}
```

### Common Animations

```css
/* Fade in */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* Scale in */
@keyframes scaleIn {
  from { transform: scale(0.95); opacity: 0; }
  to { transform: scale(1); opacity: 1; }
}

/* Slide up */
@keyframes slideUp {
  from { transform: translateY(10px); opacity: 0; }
  to { transform: translateY(0); opacity: 1; }
}
```
