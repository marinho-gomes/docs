# Typography

AI Image Generator uses the Inter font family for a clean, modern, and highly readable interface.

## Font Family

### Primary Font

```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
```

**Inter** is chosen for:

- Excellent screen readability
- Wide range of weights
- Professional appearance
- Open source availability

### Font Loading

Loaded via Google Fonts CDN:

```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
```

## Type Scale

| Name | Size | Line Height | Weight | Usage |
|------|------|-------------|--------|-------|
| XS | 11px | 1.4 | 400 | Labels, captions |
| SM | 13px | 1.4 | 400 | Secondary text, hints |
| Base | 14px | 1.5 | 400 | Body text, inputs |
| MD | 16px | 1.5 | 500 | Emphasized body |
| LG | 18px | 1.4 | 600 | Section headings |
| XL | 24px | 1.3 | 700 | Page titles |
| 2XL | 32px | 1.2 | 700 | Hero text |

## Font Weights

| Weight | Value | Usage |
|--------|-------|-------|
| Regular | 400 | Body text, descriptions |
| Medium | 500 | Emphasized text, labels |
| Semi-Bold | 600 | Subheadings, buttons |
| Bold | 700 | Headings, important |

## Text Rendering

### Anti-aliasing

For crisp text on dark backgrounds:

```css
body {
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
  text-rendering: optimizeLegibility;
}
```

### Letter Spacing

| Use Case | Value |
|----------|-------|
| Headings | -0.02em |
| Body | 0 |
| Uppercase labels | 0.05em |
| Buttons | 0.01em |

## Responsive Typography

### Mobile Adjustments

```css
@media (max-width: 480px) {
  .page-title {
    font-size: 20px;
  }

  .section-heading {
    font-size: 16px;
  }

  .body-text {
    font-size: 14px;
  }
}
```

## CSS Custom Properties

```css
:root {
  /* Font Family */
  --font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;

  /* Font Sizes */
  --font-size-xs: 11px;
  --font-size-sm: 13px;
  --font-size-base: 14px;
  --font-size-md: 16px;
  --font-size-lg: 18px;
  --font-size-xl: 24px;
  --font-size-2xl: 32px;

  /* Font Weights */
  --font-weight-regular: 400;
  --font-weight-medium: 500;
  --font-weight-semibold: 600;
  --font-weight-bold: 700;

  /* Line Heights */
  --line-height-tight: 1.2;
  --line-height-snug: 1.3;
  --line-height-normal: 1.4;
  --line-height-relaxed: 1.5;
}
```
