# Mobile App

The mobile interface provides a streamlined experience optimized for touch devices.

<Info>
  The mobile app is currently a prototype interface. Full mobile functionality is planned for future releases.
</Info>

## Home Screen

The landing screen showcases trending styles and templates.

### Hero Banner

- App title: "Image Generator"
- Tagline: "Turn your ideas into stunning images"
- Glassmorphism card effect

### Trending Styles

Horizontal scrollable carousel:

| Style | Description |
|-------|-------------|
| **Photo** | Photorealistic images |
| **Illustration** | Digital artwork |
| **Comic** | Comic book style |
| **Anime** | Japanese animation |

Tap a style to pre-select it for generation.

### Image Templates

Grid of template images showing example outputs:

- Tap to use as inspiration
- Pre-fills prompt screen with related settings

### Generate Button

Large "Generate Now" button at bottom:

- Navigates to Prompt screen
- Purple gradient styling
- Full-width design

## Prompt Screen

Configure and create your generation.

### Prompt Input

Large textarea for entering your description:

```
Example placeholder:
"A serene Japanese garden with cherry blossoms..."
```

- Pre-filled with example prompt
- Tap to edit
- Auto-resize as you type

### Random Prompt

**"Random Prompt"** button generates creative suggestions:

- Tap to fill with random creative prompt
- Good for inspiration
- Edit after generating

## Design Specifications

### Container

- Max width: 480px
- Centered on larger screens
- Full width on mobile

### Spacing

- Content padding: 20px
- Element spacing: 16px
- Card padding: 16px

### Typography

| Element | Size |
|---------|------|
| Headings | 24px |
| Body | 14px |
| Captions | 12px |

### Colors

Same dark theme as desktop:

- Background: `#0A0A0C`
- Cards: `rgba(255, 255, 255, 0.05)`
- Accent: `#8b5cf6`
- Text: `#FAFAFA`

## State Management

The app uses a simple state object:

```javascript
const appState = {
  currentScreen: "home" | "prompt" | "result",
  selectedStyle: "photo" | "anime" | "art",
  selectedShape: "square" | "landscape" | "portrait"
}
```

### Screen Transitions

- Managed by `navigateTo(screenId)` function
- Screens toggle via `.active` class
- Smooth CSS transitions

## Limitations (Prototype)

<Warning>
  Current prototype limitations:
</Warning>

- No actual AI generation (mock data)
- No persistent storage
- No user accounts
- Limited model selection
- No reference image support
